Welcome to Note Arthur!

This website is a collaboration project from multiple USF students.

The goal of this website is to be a simple note taking application.

We created this website with simplicity in mind, and our hope is that someone can use this website without any explanation and it will be intuitive enough to leave the user with no questions.

This website specifically utilizes Flask in the backend with SQLite and HTML, CSS, Javascript, and Jquery in the front-end.
